### QR Scanner App  
A tool that scans QR codes. Both web links and maps (geolocation).

#### Features:
- Launch the camera and read a QR code
- Store it in SQLite
- CRUD to SQLite - Internal Storage
- Use of maps
- Mapbox
- Markers
- Open URLs
- etc.
